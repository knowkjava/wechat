# Wechat Mini Program Integration Steps
## 1. The customer has already integrated Dynatrace. Obtain the Beacon URL and Application ID of the target Application under the Environment.
## 2. Apply for Wechat mini program authorization from [Groupstars](https://groupstars.cn/).
## 3. Import boundle.js into the mini program's root directory, modify the secret value, the Beacon URL of the Application, and the Application ID.
## 4. Modify app.ts and import boundle.js.
If you encounter any problems during integration, please raise an issue. For authorized users, please contact [Groupstars](https://groupstars.cn/)' business team: market@groupstars.cn.
